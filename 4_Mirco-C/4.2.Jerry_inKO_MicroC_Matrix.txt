2. Generating HiC contact maps using Juicer tools
# Juicer Tools is used to convert .pairs file into a HiC contact matrix.
# HiC is highly compressed binary representation of the contact matrix
# Provides rapid random access to any genomic region matrix
# Stores contact matrix at 9 different resolutions (2.5M, 1M, 500K, 250K, 100K, 
# 50K, 25K, 10K, and 5K)

## From .pairs to cooler contact matrix
# Cooler tools is used to convert indexed .pairs file into cool and mcool contact 
# matrices. Cooler generates a sparse, compressed, and binary persistent 
# representation of proximity ligation contact matrix.
# Store matrix as HDF5 file object
# Provides python API to manipulate contact matrix
# Each cooler matrix is computed at a specific resolution
# Multi-cool (mcool) files store a set of cooler files into a single HDF5 file object
# Multi-cool files are helpful for visualization

# Indexing the .pairs file
cp *.pairs .. ## make a copy of the pairs, the .pairs.gz will replace .pairs files
bgzip MicroC_inKO_Hippo_WT1_mapped.pairs
bgzip MicroC_inKO_Hippo_WT2_mapped.pairs
bgzip MicroC_inKO_Hippo_KO1_mapped.pairs
bgzip MicroC_inKO_Hippo_KO2_mapped.pairs
mv ../*.pairs .

# index the file .pairs.gz file
pairix MicroC_inKO_Hippo_WT1_mapped.pairs.gz
pairix MicroC_inKO_Hippo_WT2_mapped.pairs.gz
pairix MicroC_inKO_Hippo_KO1_mapped.pairs.gz
pairix MicroC_inKO_Hippo_KO2_mapped.pairs.gz

# Genereting single resolution contact map files
# use the cload pairix utility of Cooler to generate contact maps

cooler cload pairix -p 100 mm10.genome:1000 MicroC_inKO_Hippo_WT1_mapped.pairs.gz MicroC_inKO_Hippo_WT1_matrix_1kb.cool

cooler cload pairix -p 100 mm10.genome:1000 MicroC_inKO_Hippo_WT2_mapped.pairs.gz MicroC_inKO_Hippo_WT2_matrix_1kb.cool

cooler cload pairix -p 100 mm10.genome:1000 MicroC_inKO_Hippo_KO1_mapped.pairs.gz MicroC_inKO_Hippo_KO1_matrix_1kb.cool

cooler cload pairix -p 100 mm10.genome:1000 MicroC_inKO_Hippo_KO2_mapped.pairs.gz MicroC_inKO_Hippo_KO2_matrix_1kb.cool



# Genereting multi-resolutions files and visualizing the contact matrix
# When you wish to visualize the contact matrix, it is highly recommended to generate 
# a multi-resolution .mcool file to allow zooming in and out to inspect regions of 
# interest. The cooler zoomify utility allows you to generate a multi-resolution 
# cooler file by coarsening. The input to cooler zoomify is a single resolution .cool 
# file, to allow zooming in into regoins of interest we suggest to generate a .cool 
# file with a small bin size, e.g. 1kb. Multi-resolution files uses the suffix .mcool.
# resolutions (kb): 1 2 4 8 16 32 64 128 256 512 1024 2048 4096 8192
cooler zoomify --balance -p 100 MicroC_inKO_Hippo_WT1_matrix_1kb.cool 
cooler zoomify --balance -p 100 MicroC_inKO_Hippo_WT2_matrix_1kb.cool
cooler zoomify --balance -p 100 MicroC_inKO_Hippo_KO1_matrix_1kb.cool
cooler zoomify --balance -p 100 MicroC_inKO_Hippo_KO2_matrix_1kb.cool
